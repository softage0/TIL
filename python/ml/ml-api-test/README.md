## How to setup

```bash
$ pip install -r requirements.txt # on venv environment if possible
```


## How to run API server

```bash
$ python api.py
```


## How to use

request `http://localhost:5000/ann/{x_data}` from browser or curl command

ex)
```
http://localhost:5000/ann/[[0,0],[1,0],[1,1],[0,0],[0,0],[0,1]]
```
